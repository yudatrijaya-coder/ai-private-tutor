## 1. Design System

```css

  --st-secondary: #f97316;     /* Orange вҖ” streaks, fire */

  --st-purple:    #a855f7;     /* Purple вҖ” special, premium */

```

### 1.2 Typography

| Role | Font | Weight | Usage |
|------|------|--------|-------|
| Display | `Outfit` / `Plus Jakarta Sans` | 700-800 | Headings, big numbers |
| Body | `Inter` | 400-600 | All text |
| Mono | `JetBrains Mono` / `Fira Code` | 400 | Agent logs, JSON |
| Student Display | `Fredoka` / `Nunito` | 600-700 | Student headings вҖ” rounded & friendly |
| Student Body | `Nunito` / `Inter` | 400-600 | Student text вҖ” rounded terminals |

**Why 2 display fonts?** Superuser = sharp/serious. Student = round/playful. Both work at every size.

### 1.3 Spacing & Radius

```css
/* Shared */
--radius-sm: 6px;
--radius-md: 12px;
--radius-lg: 20px;
--radius-xl: 28px;

/* Student-only: bigger, more pill-shaped */
--radius-card: 20px;
--radius-button: 999px; /* Pill buttons вҖ” friendly */
--radius-avatar: 999px;
```

## 5. Key UX Principles

### 5.1 Superuser Dashboard
- **Dark mode first** вҖ” professional, easy on eyes for long monitoring
- **Data density > whitespace** вҖ” colok mata, informasi sebanyak mungkin
- **Color = meaning** вҖ” no random colors. Green/amber/red = status
- **Progressive disclosure** вҖ” overview вҶ’ click вҶ’ detail drawer

### 5.2 Student App
- **Light + warm** вҖ” cozy, gak serem, pengen balik lagi
- **Big everything** вҖ” buttons, cards, fonts (child motor skill & readability)
- **Safe failure** вҖ” wrong answer = encouraging, not punishing (Duolingo pattern)
- **Visible progress everywhere** вҖ” XP bar, streak, mastery bar selalu keliatan
- **Mascot as emotional interface** вҖ” bukan hiasan, tapi relationship vector
- **Low friction** вҖ” tap 2x udah bisa belajar, gak perlu form panjang
- **Bottom nav** вҖ” thumb-friendly, gak perlu reach atas
